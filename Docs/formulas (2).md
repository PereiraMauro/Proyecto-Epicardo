# formulas 

## Fórmulas de cálculo

-   Precio Base: `PB`
-   IVA (21%): `IVA = PB * 0.21`
-   Impuesto PAÍS (30%): `PAIS = PB * 0.30`
-   Percepción RG 4815 (45%): `RG4815 = PB * 0.45`
-   Percepción RG 5403 (5%): `RG5403 = PB * 0.05`

### Precio Final
    PF = PB + IVA + PAIS + RG4815 + RG5403
ó
    PF = PB * 2.01

## Ejemplo
Si un juego cuesta **10 USD**:

    IVA = 10 * 0.21 = 2.1
    PAIS = 10 * 0.30 = 3.0
    RG4815 = 10 * 0.45 = 4.5
    RG5403 = 10 * 0.05 = 0.5

    PF = 20.1 USD
